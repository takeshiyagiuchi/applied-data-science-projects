# Applied Data Science Projects

This repository contains a collection of applied data science projects focused on real-world prediction and analysis tasks. Each project is self-contained and includes data exploration, modeling, and an executive summary of results.

---

## Projects

### 🏠 House Price Prediction
Predicting residential property prices using regression models and feature engineering techniques.

- Models: Linear Regression  
- Focus: Feature engineering and model generalization (overfitting vs. underfitting)  
- 👉 [View Project](house-price-prediction/)

---

### 🚗 Insurance Quote Prediction
Predicting whether a car insurance quote is bound using classification and regression approaches, with a focus on business-oriented evaluation metrics.

- Models: Random Forest, Logistic Regression  
- Focus: Feature engineering, regularization, hyperparameter tuning, ROC-AUC, and threshold optimization  
- 👉 [View Project](car-insurance-prediction/)
